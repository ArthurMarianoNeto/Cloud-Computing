Iniciando

Para começar a usar este CD, você precisa instalar os pacotes de software. Antes de começar, no entanto, assegure-se de que seu computador atende aos Requisitos de Sistema a seguir.

Requisitos de Sistema

Sistema operacional

Para poder usar tudo neste CD, você precisa estar rodando o Linux. O Red Hat Linux 6.2 é a plataforma Linux suportada oficialmente.

Você também pode tentar usar o Mandrake Linux 7.1, o Turbo Linux Workstation 6.0 ou o SuSe Linux 6.4.

Se instalar uma versão mais recente de glibc (v2.1.2-11 ou posterior), você também pode experimentar o Caldera OpenLinux 2.4, embora, por ser fornecido com a glibc v2.1.2-3, ele não seja compatível com o Java 2 SDK.

Você pode tentar usar o Red Hat Linux 6.1 ou 7.1, mas o Red Hat Linux 7.0 não é compatível com o Java 2 SDK.

Processador

Para usar tudo, você precisa ter, no mínimo, um processador Pentium/compatível de 166 MHz (mínimo) ou mais rápido. O Forte for Java exige um processador de, no mínimo, 350 MHz.

Placa de vídeo gráfica / Placa de som

Uma placa de vídeo gráfica padrão é adequada. Para reprodução de áudio com o Java Media Framework, se sua máquina Linux não tiver suporte a áudio embutido, você precisa de uma placa de som.

RAM

Para usar tudo no Linux, você precisa de, no mínimo, 128 MB de RAM, mas recomenda-se 256 MB.

Espaço em disco

Para instalar tudo o que está no CD, recomenda-se que você tenha pelo menos 600 MB de espaço disponível em disco rígido.

IMPORTANTE!

Requisitos de ambiente de software

Para instalar e usar diversos programas incluídos neste CD-ROM, você deve ter certos programas que são pré-requisitos instalados. Na maioria dos casos, este software é fornecido como pacote de software separado ou como componente de um dos pacotes. Alguns dos requisitos, entretanto, não estão incluídos no CD.

A tabela a seguir indica as dependências dos diversos programas de software entre eles e em relação a outros programas. Você deve ler esta tabela agora para obter uma visão geral e relê-la antes de instalar qualquer software deste CD-ROM. Volte a esta tabela novamente para depurar quaisquer problemas de instalação que possam surgir. Leia as anotações que fornecem mais detalhes sobre muitos destes requisitos. Em cada caso, é altamente recomendado que você veja a homepage dos dois programas para determinar o que mudou entre a gravação do CD e sua compra do livro-texto.

Instalando o Software

Agora que você verificou que o seu sistema atende aos requisitos mínimos para instalar o software, você deve ler os contratos de licenciamento. Leia o texto completo de cada um dos contratos de licenciamento e certifique-se de que você concorda com eles. Agora você está pronto para instalar. Para obter instruções sobre como instalar cada um dos pacotes de software, clique aqui.

Exemplos

Depois que você tiver instalado todo o software, você estará pronto para começar a trabalhar com os exercícios ao longo do livro. O diretório "exemplos" pode ser visualizado clicando-se aqui.
