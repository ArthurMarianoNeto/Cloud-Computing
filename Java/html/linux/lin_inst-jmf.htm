Java™ Media Framework
para instalação em Linux

Leia a documentação a seguir antes de você instalar o Java™ Media Framework a partir do CD.

Você leu o contrato de licenciamento do Java™ Media Framework?

Requisitos

Blackdown ou Sun Java2 Standard Edition SDK ou Runtime Environment v1.3.0 ou mais recente para Linux/i386

Instalação

Você precisa instalar somente um dos pacotes a seguir. Os dois arquivos estão localizados no diretório software-lin\framework no nível do diretório-raiz do CD-ROM.

  • Pacote padrão: jmf-2.1.1-fcs-linux-i386.tar.bz2

    • Baixe jmf-2.1.1-fcs-linux-i386.tar.bz2 do CD para seu disco rígido, usando sua ferramenta gráfica ou de linha de comando preferida, depois siga as instruções abaixo.

    Instalação:

    % cd <targetdirectory>
    % bzip2 -dc <downloadpath>/jmf-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
    % export PATH=<targetdirectory>/jmf-2.1.1/bin:$PATH
    % export CLASSPATH=<targetdirectory>/jmf-2.1.1/lib/jmf.jar:<targetdirectory>/jmf-2.1.1/lib/sound.jar:<targetdirectory>/jmf-2.1.1/lib/mediaplayer.jar:<targetdirectory>/jmf-2.1.1/lib/multiplayer.jar:<targetdirectory>/jmf-2.1.1/lib:$CLASSPATH
    % export LD_LIBRARY_PATH=<targetdirectory>/jmf-2.1.1/lib:$LD_LIBRARY_PATH

  • Pacote de extensão de Java2: jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2

    • Baixe jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 do CD para seu disco rígido, usando sua ferramenta gráfica ou de linha de comando favorita, depois siga as instruções abaixo.

    • Instalação com um J2SDK 1.3:
      • % cd <j2sdk1.3.0 installation>/jre
      % bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
      % export PATH=<j2sdk1.3.0 installation>/jre/bin:$PATH
    • Instalação com um J2RE 1.3:
      • % cd <j2re1.3.0 installation>
      % bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf-

Data Integrity Error

Se você estiver extraindo o arquivo compactado com a linha de comando e receber uma mensagem de erro que diz "Data integrity error when decompressing", faça uma das seguintes alternativas:

  1. Verifique a localização do diretório para o qual você estava expandindo o arquivo compactado. O diretório expandido pode estar intacto.

  2. Use um utilitário de descompactação com GUI, como o "ark" para expandir o arquivo compactado.
