Iniciando

Para começar a usar este CD, você precisa instalar os pacotes de software. Antes de começar, no entanto, assegure-se de que seu computador atende aos seguintes requisitos de sistema.

Requisitos de Sistema

Sistema operacional

Para poder usar tudo neste CD, você precisa estar rodando o Windows 95, o Windows 98, o Windows NT ou o Windows 2000.

Processador

Para usar tudo, você precisa ter no mínimo um processador Pentium/compatível de 166 MHz (mínimo) ou mais rápido. O Forte for Java exige um processador de, no mínimo, 350 MHz.

Placa de vídeo gráfica/Placa de som

Uma placa de vídeo gráfica padrão é adequada. Para reprodução de áudio com a Java Media Framework, se sua máquina Windows não tiver suporte a áudio embutido, você precisa de uma placa de som.

RAM

Para usar tudo no Microsoft Windows, você precisa de, no mínimo, 128 MB de RAM, mas recomenda-se 256 MB.

Espaço em disco

Para instalar tudo o que está no CD, recomenda-se que você tenha pelo menos 800 MB de espaço disponível em disco rígido.

Alguns materiais neste CD podem exigir o uso de um utilitário de compactação de arquivos Zip. Se não tiver um, você pode baixar uma versão grátis para avaliação do WinZip em http://www.winzip.com

Instalando o Software

Agora que você verificou se o seu sistema atende aos requisitos mínimos para instalar o software, você deve ler os contratos de licenciamento. Leia o texto completo de cada um dos contratos de licenciamento e certifique-se de que você concorda com eles. Agora você está pronto para instalar. Para obter instruções sobre como instalar cada um dos pacotes de software, clique aqui.

Exemplos

Depois de instalar todo o software, você está pronto para começar a trabalhar com os exercícios ao longo do livro. Pode-se visualizar o diretório "exemplos" clicando-se diretamente no botão Exemplos na tela inicial ou clicando-se aqui.

Isto o levará para uma janela que mostra os diretórios de exemplos do CD. Eles estão organizados por número de capítulo e contêm o código dos exemplos para cada capítulo em particular. Você pode copiá-los para seu disco rígido e modificá-los à medida que você progredir ao longo do livro.

Links

Para obter mais informações a partir da Internet, baixar as atualizações de software dos sites dos fornecedores e aprender mais sobre o fantástico mundo da World Wide Web, clique aqui.

Para obter os exemplos de código e os links ativos WWW mais atualizados, visite o endereço http://www.deitel.com/

NOTA: Você precisa ter uma conexão ativa com a Internet para usar os links.
