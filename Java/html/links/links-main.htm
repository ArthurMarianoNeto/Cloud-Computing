Um outro site útil na Web é JARS—originalmente chamado de Java Applet Rating Service. O site JARS se auto denomina o "#1 Java Review Service." Este site originalmente era um enorme repositório de applets Java. JARS agora é um recurso abrangente para programadores Java.

Grupos de notícias podem se constituir em um recurso valioso para quem está aprendendo Java ou para quem tem perguntas sobre tópicos específicos. Ao colocar suas próprias perguntas em um grupo de notícias, forneça detalhes específicos do problema que você está tentando resolver (tal como um problema com um programa que você está escrevendo). Isto irá permitir a outras pessoas que lerem o grupo de notícias entender sua colocação e (espera-se) dar uma resposta a você. Certifique-se de especificar um título de assunto que claramente indique seu problema. Se você estiver respondendo a uma pergunta de outra pessoa, verifique sua resposta antes de remetê-la, para assegurar que está correta. Grupos de notícias não devem ser usados para promover produtos ou serviços, nem as informações de contato (como endereços de e-mail de outros usuários do grupo de notícias) devem ser usadas para propósitos não relacionados. Em geral, estes não são fóruns para bate-papo, de modo que as publicações devem ser corteses e ir direto ao assunto.
Normalmente, um software para leitura de grupos de notícias é necessário para você interagir com um grupos de notícias. Este tipo de software é fornecido como parte do Netscape Navigator e do Microsoft Outlook Express e existem muitos outros programas para grupos de notícias disponíveis. Você também pode acessar grupos de notícias através da Web. Se você ainda não usa grupos de notícias, experimente o site Google.
groups.google.com
Digite o nome do grupo que você gostaria de ver (você também pode pesquisar por grupos de notícias) e você receberá uma lista dos tópicos e perguntas atuais daquele grupo de notícias. Abaixo estão relacionados alguns grupos de notícias sobre Java que você pode achar úteis.
news:comp.lang.java.advocacy
Este grupo é um centro ativo de discussões sobre a cultura Java atual, incluindo os méritos de diferentes linguagens de programação.
news:comp.lang.java.announce
Este grupo faz a divulgação das principais extensões de Java, novas bibliotecas de classes e conferências.
news:comp.lang.java.api
Este grupos contém perguntas sobre bugs, erros de compilação, especificações de Java e quais classes são mais apropriadas para diferentes situações.
news:comp.lang.java.gui
Este grupo responde a problemas encontrados ao trabalhar com interfaces gráficas com usuário em Java. Se você tem problemas com um componente particular, leiaute ou evento, este pode ser um bom lugar para iniciar.
news:comp.lang.java.help
Este grupo é particularmente ativo. Ele aborda muitas questões sobre linguagem e ambiente. Colocações incluem solicitações de classes o algoritmos para resolver um problema específico.
news:comp.lang.java.machine
Para pessoas interessadas no funcionamento interno de Java, este grupo se concentra na Java Virtual Machine.
news:comp.lang.java.misc
Este grupo contém de tudo, de ofertas de emprego a perguntas sobre a documentação Java, e se destina a colocações que não se enquadram em outras categorias de grupo de notícias sobre Java.
news:comp.lang.java.programmer
Este grupo é outro fórum extremamente ativo, que trata de uma variedade de perguntas. Colocações tendem a ser orientadas a projetos e com preocupações sobre estilo geral de programação e estrutura.
news:comp.lang.java.softwaretools
Este grupo de notícias é centrado em produtos de software escritos em Java, seus usos, suas falhas e possíveis modificações. Algumas perguntas sobre escrita de software eficiente também são encontradas aqui.
news:comp.lang.java.tech
Este é um grupo de notícias devotado a aspectos técnicos de Java e seu funcionamento interno.
